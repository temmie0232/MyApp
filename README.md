# Todo_App_login

ログイン機能により、プライベートなTodo管理をブラウザから行えます

## Objective

- echo や gin などのフレームワークなどは使用せずに、標準パッケージやサードパッケージのみでユーザーの登録、ログイン、ログアウト機能や、データのCRUD処理（net/http）を実装しました。
- heroku にデプロイ。
- Goのシステム開発の上での基盤の理解（応用可能）

## Specification

- Golang, HTML, CSS, Bootstrap, jQuery
